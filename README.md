# Python_homework
现代编程技术大作业
===
主要功能
-
    1.哈希函数
        可以将输入的数值转为哈希数值，可以看作是一种加密方式
    2.写“大”字程序
        利用turtle库写“大”字，可调整字号和颜色
    3.七段数码管显示
         (1)显示输入的日期
         (2)显示当前系统时间
         (3)显示任意数字
    4.显示文本中的保留字
        将文本(Fault.txt)中的保留字以字典形式输出键值对
    5.模拟A和B的竞技比赛
        引入天气、能力值、基础失误率等因素
    6.读同一个页面中的多个URL
        利用递归算法输出设定深度的url
    7.语法检测器
        可检查for、try_except、if_elif_else
